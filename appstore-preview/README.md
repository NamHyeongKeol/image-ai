# appstore-preview

React + TypeScript + Tailwind + shadcn 스타일 기반의 iPhone App Store 프리뷰 합성 도구입니다.

## 핵심 기능

- 고정 캔버스: `886x1920`
- 이미지/영상 업로드 후 iPhone 화면 영역에 합성
- 배경 단색/그라데이션, 각도 조절
- 상단 텍스트 입력 + 서체/색상/크기 변경
- 입력이 이미지면 PNG 출력, 입력이 영상이면 영상 출력

## 실행

```bash
npm install
npm run dev
```

## 빌드/검증

```bash
npm run lint
npm run build
```
